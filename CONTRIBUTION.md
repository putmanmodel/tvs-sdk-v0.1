At this stage, the project is not accepting open code contributions through pull requests by default.

**Stephen A. Putman**  
**Email:** putmanmodel@pm.me

## Bug reports

If you find a bug, implementation issue, documentation problem, or unclear behavior, please report it by email.

When possible, include:

- a clear subject line
- a short description of the issue
- reproduction steps
- expected behavior
- actual behavior
- environment details, if relevant

Please use a subject line such as:

- `TVS SDK Bug Report`
- `TVS SDK Documentation Issue`
- `TVS SDK Contribution Inquiry`
- `TVS SDK Licensing Inquiry`
